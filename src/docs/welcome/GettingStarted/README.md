This is the Addon Panel which supports markdown files, knobs, and action loggers. 

Keyboard Shortcuts

⌘ ⇧ O/  ⌃ ⇧ O `Show Search Box`

⌘ ⇧ G/  ⌃ ⇧ G `Toggle Addon panel position`

⌘ ⇧ F/  ⌃ ⇧ F `Toggle Fullscreen Mode`

⌘ ⇧ X/  ⌃ ⇧ X `Toggle Stories Panel`

⌘ ⇧ C/  ⌃ ⇧ C `Toggle Addon panel`

⌘ ⇧ →/  ⌃ ⇧ → `Next Story`

⌘ ⇧ ←/  ⌃ ⇧ ← `Previous Story`
